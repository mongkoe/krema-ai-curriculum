    0. 개관

    생성 AI 발전 현황 & 업무 적용 지점

    • 텍스트를 넘어선 멀티모달 — 글·이미지·표·코드·음성·영상을 한 도구에서 생성하는 시대
    • “대화”가 인터페이스 — 함수·매크로·디자인 툴을 몰라도 말로 시켜서 결과물을 얻는다
    • 마케팅 기획 업무에 바로 적용
      • 리서치 — 정책·시장·경쟁사 자료를 빠르게 수집·요약
      • 문서 — 회의록·제안서·보고서 초안을 분 단위로
      • 데이터 — 매출·VOC·공급물량을 분석하고 차트로 시각화
    관점 전환 — “AI가 내 일을 대신한다”가 아니라, AI를 잘 시키는 사람이 같은 시간에 5배를 한다.

    1-1생성 AI란 무엇인가

    대량의 데이터로 학습해, 다음에 올 말을 확률적으로 “생성”하는 AI

    🔎기존 검색

    이미 있는 문서를 찾아서 링크로 보여준다 (검색 결과 나열)

    ✍️생성 AI

    학습한 패턴으로 새 문장·표·코드를 직접 만들어 답으로 준다

    • LLM(거대 언어모델) — ChatGPT의 두뇌. 말의 “다음 단어”를 끝없이 예측해 문장을 완성
    • 핵심은 ‘맥락 제공’ — 같은 두뇌라도 내가 주는 정보·지시에 따라 결과 품질이 완전히 갈린다
    한 줄 요약 — 검색은 ‘찾기’, 생성 AI는 ‘만들기’. 그래서 시키는 법(프롬프트)이 전부다.

    1-2생성 AI 실제 활용 사례

    마케팅 기획 실무에서 오늘 당장 쓰는 장면들

    📝문서

    회의록 정리, 제안서·기획서 초안, 이메일·공지문 작성

    📈분석

    매출·재고·설문 데이터 요약, 인사이트 도출, 차트 생성

    💡기획

    네이밍·슬로건 후보, 캠페인 아이디어, 타깃 페르소나 정의

    🔬리서치

    정책·규제 요약, 경쟁 단지 비교, 시장 동향 정리

    🌐번역·교정

    외국 자료 번역, 맞춤법·톤 교정, 길이 조절

    🗂️요약

    긴 보고서·논문·회의 녹취를 3줄/표로 압축

    실습 예고 — 섹션 2에서 회의록·제안서·정량/정성 데이터 분석을 직접 만들어 봅니다.

    1-3ChatGPT 화면 구성과 사용법

    처음 켜면 보이는 요소들 — 어디에 무엇이 있나

    • 왼쪽 사이드바 — 대화 목록, 새 채팅, Projects(폴더), 검색
    • 가운데 입력창 — 프롬프트 입력. + 버튼으로 파일·이미지·PDF 첨부
    • 상단 모델 선택 — GPT-5 / 추론 모델 등 작업에 맞춰 전환
    • 도구 메뉴 — 웹 검색, 이미지 생성, 캔버스(Canvas), 데이터 분석(코드 실행)
  • 새 채팅 열기 → 모델 확인
  • 필요하면 자료 첨부(+)
  • 프롬프트 입력 → Enter
  • 답이 부족하면 이어서 수정 요청(한 번에 끝내려 하지 않기)
    • Tip — 줄바꿈은 Shift+Enter. 바로 Enter는 전송됩니다.

    1-4프롬프트 엔지니어 추천 기본 세팅

    한 번 해두면 모든 대화가 좋아지는 설정

    • 맞춤 설정(Custom Instructions) — 내 직무·회사·말투를 미리 등록 → 매번 설명 안 해도 됨
      • “나는 부동산 분양 마케팅 기획자다. 답은 한국어, 표·불릿 위주, 결론부터.”
    • 메모리(Memory) 켜기 — 자주 쓰는 맥락을 기억하게
    • 언어·톤 고정 — “전문적이되 간결하게, 과장 없이”
    • 작업별 모델 습관 — 빠른 답은 기본 모델, 깊은 분석·기획은 추론 모델
    맞춤 설정 예시 (설정 ▸ 맞춤 설정)나는 부동산 분양 마케팅 기획자입니다. - 모든 답변은 한국어, 결론 먼저, 표/불릿 활용 - 추측은 '추정'이라 표시하고 근거를 함께 - 과장·홍보성 수식어 배제, 실무 톤 유지

    1-5ChatGPT는 어떻게 답을 만드나

    원리를 알면 ‘왜 틀리는지’와 ‘어떻게 잘 시킬지’가 보인다

    • 다음 단어 예측 — 문맥을 보고 가장 그럴듯한 다음 토큰을 하나씩 이어 붙인다
    • ‘이해’가 아니라 ‘패턴’ — 사실 DB를 조회하는 게 아니라 학습된 확률로 생성 → 그래서 틀린 사실도 자신 있게 말할 수 있다
    • 입력(맥락)이 길수록 좋아진다 — 배경·예시·형식을 줄수록 예측이 정확해진다
    • 최신성 한계 — 학습 시점 이후 정보는 모를 수 있다 → 웹 검색 도구로 보완
    그래서 — ① 맥락을 충분히 주고 ② 사실은 검증하고 ③ 형식을 지정하면 품질이 급상승한다. 이게 바로 1-6 프롬프트 3단 구조.
    핵심 ★

    1-6프롬프트 작성 3단 구조

    오늘 단 하나만 가져간다면 이것 — 역할 · 맥락/지시 · 형식

    ①역할 부여

    “너는 ○○ 전문가다” — 답변의 관점·전문성을 고정

    ②맥락 + 지시

    배경 정보 + 무엇을 / 누구를 위해 / 조건은

    ③출력 형식

    표·불릿·글자수·톤·예시 — 결과물 모양을 지정

    3단 구조 적용 예시① 너는 15년 차 분양 마케팅 카피라이터다. ② 강남 신축 오피스텔(1.5룸, 1인 가구 타깃)의 옥외광고 헤드라인이 필요해. 핵심 강점은 '역세권 도보 3분'과 '풀옵션'이다. ③ 8자 이내 헤드라인 5개를, 톤이 다른 버전으로 표(번호·문구·의도)로 만들어줘.
    기억법 — “누구로서 / 무엇을 / 어떤 모양으로”. 답이 마음에 안 들면 셋 중 빠진 걸 채워라.

    1-7할루시네이션(환각) 주의와 예방

    그럴듯하게 지어낸 거짓 정보 — 가장 흔한 사고 원인

    ⚠️이런 게 환각

    • 존재하지 않는 통계·법조항·논문 인용
    • 실거래가·날짜·수치를 ‘대충’ 만들어냄
    • 없는 출처를 진짜처럼 표기

    🛡️예방법

    • “근거·출처를 함께, 모르면 모른다고”
    • 숫자·법령은 원문/공식 사이트로 교차 확인
    • 자료를 첨부해서 “이 안에서만 답해”
    환각을 줄이는 한 줄확실한 사실만 답하고, 불확실하면 "확인 필요"라고 표시해줘. 추측이 포함되면 어디가 추측인지 명시하고, 출처를 함께 제시해줘.
    실무 원칙 — AI 답변은 초안이다. 외부에 나가는 숫자·법규는 사람이 검증한다.

    1-8ChatGPT 고급 기능

    ‘채팅’ 너머의 진짜 생산성 도구들

    • 파일 업로드 분석 — 엑셀·PDF·이미지를 올려 요약·분석·표 추출
    • 데이터 분석(코드 실행) — 업로드한 표로 직접 계산·차트 생성
    • 웹 검색 — 최신 정보 조회 후 출처와 함께 답변
    • 캔버스(Canvas) — 긴 문서/코드를 옆 패널에서 함께 편집
    • 이미지 생성 — 텍스트로 이미지·시안 제작
    • 음성 대화 — 말로 묻고 듣기
    오늘 집중 — 파일 분석 + 데이터 분석 + 웹 검색. 섹션 2·3 실습에서 직접 씁니다.

    ChatGPT 활용 다양한 문서 작성

    실무 문서 4종을 직접 만들어 봅니다 (2-1 ~ 2-4)

    2-1 회의록 2-2 영업 제안서 2-3 정량 데이터 분석 2-4 VOC 정성 분석·시각화
    실습 ①

    2-1회의록 작성하기

    두서없는 녹취·메모 → 깔끔한 회의록

  • 회의 녹취/메모를 그대로 붙여넣기 (또는 음성파일 첨부)
  • 역할·형식 지정해 정리 요청
  • 결정사항·담당자·기한을 표로 추출
    • 프롬프트너는 꼼꼼한 회의 서기다. 아래 회의 메모를 회의록으로 정리해줘. [형식] 1) 일시·참석자 2) 안건별 논의 요약 3) 결정사항 4) 액션아이템(담당자·기한)을 표로 [메모] (여기에 회의 내용 붙여넣기)
    Tip — 음성파일을 첨부하면 받아쓰기까지 한 번에. 정리 후 “3줄 요약도 추가”라고 이어서 요청.
    실습 ②

    2-2자사 서비스 기반 영업 제안서

    우리 서비스 자료를 먹이고 → 고객 맞춤 제안서로

  • 자사 소개서·서비스 자료(PDF)를 첨부
  • 타깃 고객·니즈·제안 목적을 명시
  • 제안서 구조(목차)대로 초안 생성 → 표현 다듬기
    • 프롬프트너는 B2B 영업 제안 전문가다. 첨부한 자사 서비스 자료를 근거로 [고객] 신규 분양 시행사 / [니즈] 미분양 리스크 최소화 [목적] 분양 마케팅 대행 수주 위 고객을 위한 제안서 초안을 작성해줘. [형식] 1)고객 과제 2)우리 솔루션 3)기대효과 4)추진 일정 5)견적 개요
    핵심 — 자료를 첨부해야 환각 없이 ‘우리 것’으로 쓴다. 빈손으로 시키면 일반론만 나온다.
    실습 ③

    2-3매출·재고 정량 데이터 분석

    엑셀을 올리고, 질문으로 분석시키기 (데이터 분석 기능)

  • 매출/재고 엑셀(.xlsx) 업로드
  • 알고 싶은 질문을 자연어로
  • 요약·이상치·추세 분석 + 차트 생성
    • 프롬프트첨부한 매출 데이터를 분석해줘. 1) 월별·제품별 매출 추세를 요약 2) 전월 대비 급감/급증 항목과 그 의미 3) 재고 회전이 느린 상위 5개 품목 4) 핵심 인사이트 3가지와 차트(막대·선)로 시각화
    주의 — 숫자 결과는 한 번 검산. 합계·기간 기준이 맞는지 “계산 근거도 보여줘”로 확인.
    실습 ④

    2-4VOC 정성 데이터 분석·시각화·FAQ

    고객의 말(리뷰·문의) → 분류·감정·FAQ로 구조화

  • VOC(리뷰/문의/설문 주관식) 텍스트 또는 파일 입력
  • 주제별 분류 + 긍·부정 감정 분석
  • 자주 묻는 질문 → FAQ 초안 + 시각화
    • 프롬프트아래는 분양 상담 고객의 문의/불만 모음이다. 1) 주제별로 분류(가격·입지·계약·AS 등)하고 건수 표로 2) 긍정/부정 비율과 대표 의견 3) 상위 문의 10개로 FAQ 초안 작성 4) 주제별 비중을 차트로 시각화 [데이터] (붙여넣기)
    03

    ChatGPT Project 기능 활용 보고서 작성

    자료를 한 폴더에 모아두고 ‘맥락을 기억하는’ 작업방 (3-1 ~ 3-5)

    3-1 프로젝트란 3-2 vs 일반 대화 3-3 사용법·주의 3-4 정책보고서 실습 3-5 Deep Research

    3-1프로젝트란 무엇인가

    관련 대화 + 자료 + 지침을 묶는 ‘업무방’

    • 공통 지침 — 그 방의 모든 대화에 적용되는 역할·규칙을 한 번만 설정
    • 공통 파일 — 자료를 방에 올려두면 모든 대화가 그 자료를 참고
    • 대화 묶음 — 한 주제의 여러 대화를 한 곳에 정리 → 맥락 유지
    비유 — 일반 대화가 ‘낱장 메모’라면, 프로젝트는 자료·규칙이 꽂힌 바인더. 매번 배경 설명을 반복하지 않아도 된다.

    3-2프로젝트 vs 일반 대화

    구분일반 대화프로젝트
    맥락 유지대화 1개 안에서만여러 대화가 공유
    자료그때그때 첨부방에 상시 비치
    지침매번 다시 설명한 번 설정 → 자동 적용
    적합단발성 질문지속 프로젝트·보고서
    언제 쓰나 — “이 단지 분양 캠페인”, “2026 시장조사”처럼 며칠~몇 주 이어지는 일은 프로젝트로.

    3-3프로젝트 사용 방법과 주의사항

  • 사이드바에서 + New Project 생성·이름 지정
  • 프로젝트 지침(Instructions)에 역할·규칙 입력
  • 참고 파일 업로드(정책 PDF, 데이터 등)
  • 그 안에서 대화 시작 → 자료·지침이 자동 반영

    • ✅좋은 습관

    • 주제 1개 = 프로젝트 1개
    • 자료 최신본으로 갱신

    ⚠️주의

    • 민감·기밀 자료 업로드 정책 확인
    • 오래된 파일 방치 → 잘못된 근거
    실습

    3-4부동산 정책보고서 작성 프로젝트

    정책 원문을 넣고 → 우리 사업 관점의 보고서로

  • 프로젝트 생성 + 지침에 “부동산 정책 분석가” 역할 설정
  • 정책 자료(보도자료·고시·매뉴얼 PDF) 업로드
  • 요약 → 영향 분석 → 대응 전략 순으로 대화
    • 프로젝트 지침너는 부동산 정책 분석가다. 업로드된 자료에 근거해서만 답하고, 추정은 '추정'으로 표시한다. 출력은 결론-근거-시사점 순.
    대화 프롬프트업로드한 정책 자료를 바탕으로 보고서를 작성해줘. 1) 정책 핵심 3줄 요약 2) 분양 시장에 미치는 영향 3) 우리 사업(분양 마케팅) 관점의 기회/리스크 4) 대응 제언

    3-5Deep Research (심층 리서치)

    AI가 여러 출처를 스스로 검색·종합해 ‘리포트’를 써준다

    • 일반 검색과 차이 — 수십 개 출처를 다단계로 조사 → 출처 포함 장문 보고서
    • 시간 — 수 분 소요(그만큼 깊다). 주제·범위·기간을 구체적으로 줄수록 좋다
    • 마케팅 활용 — 시장 동향, 경쟁 단지 분석, 정책 변화 종합 리서치
    프롬프트'2025~2026 수도권 분양 시장 동향'을 심층 조사해줘. - 공급물량·청약경쟁률·미분양 추이 / 가격 흐름 / 정책 변수 - 신뢰할 수 있는 출처(정부·통계·언론)만, 인용 표기 - 마지막에 마케팅 시사점 5가지
    검증 — 심층 리포트도 출처 클릭해 핵심 수치는 원문 확인.
    04

    다양한 생성 AI ① — 검색·요약

    ChatGPT 밖의 무기들: 출처 기반 검색과 자료 요약 (4-1 ~ 4-3)

    4-1 Perplexity 4-2 NotebookLM 4-3 SciSpace

    4-1Perplexity

    ‘출처가 보이는’ AI 검색 — 리서치의 시작점

    ⭐강점

    • 답변마다 출처 링크 표기 → 신뢰·검증 쉬움
    • 최신 웹 정보 반영
    • 후속 질문으로 파고들기

    🎯이럴 때

    • “근거 있는” 시장·정책 조사
    • 경쟁사·통계 빠른 확인
    • 회의 전 5분 사전 조사
    프롬프트2026년 수도권 아파트 미분양 현황과 원인을 출처와 함께 정리해줘. 지역별 수치와 최근 기사 링크를 표로.
    섹션 6 예고 — Perplexity로 ‘주택 공급 물량’을 직접 분석합니다.

    4-2NotebookLM

    내가 올린 자료‘만’ 근거로 답하는 구글의 리서치 도구

    ⭐강점

    • 업로드한 문서 안에서만 답 → 환각 최소
    • 긴 PDF·여러 문서 동시 분석
    • 오디오 요약(팟캐스트) 생성

    🎯이럴 때

    • 매뉴얼·계약서·정책 원문 정독
    • 방대한 자료를 Q&A로
    • 근거 출처(원문 인용) 필요할 때
    사용법1) 노트북 생성 → 소스로 PDF·웹·문서 업로드 2) 질문: "이 매뉴얼에서 인허가 절차를 순서대로 정리해줘" 3) 답변의 인용 번호 클릭 → 원문 위치 확인
    섹션 6 예고 — 신속통합기획 매뉴얼을 NotebookLM으로 분석합니다.

    4-3SciSpace

    논문·학술 자료 검색과 이해 특화

    ⭐강점

    • 논문 검색 + 핵심 요약
    • 어려운 내용 쉬운 설명
    • 여러 논문 비교 표

    🎯이럴 때

    • 부동산·도시·소비자 행동 연구 근거
    • 보고서에 학술적 신뢰 더하기
    • 트렌드의 ‘이론적 배경’ 찾기
    프롬프트'주거 선택에 영향을 미치는 입지 요인' 관련 논문을 찾아 핵심 발견과 시사점을 표로 요약하고, 마케팅 적용점을 덧붙여줘.

    다양한 생성 AI ② — 데이터 시각화

    숫자·개념을 ‘보이게’ 만드는 도구들 (5-1 ~ 5-3)

    5-1 Claude Artifacts 5-2 Gemini 5-3 Napkin AI

    5-1Claude AI — Artifacts

    대화 옆에서 ‘작동하는’ 차트·문서·웹페이지를 즉석 생성

    ⭐강점

    • 인터랙티브 차트·대시보드를 코드 없이
    • 긴 글쓰기·문서 품질 우수
    • 결과물을 바로 미리보기·수정

    🎯이럴 때

    • 데이터를 보여주는 시각자료
    • 제안서용 인포그래픽·표
    • 간단한 계산기·웹 목업
    프롬프트아래 분기별 분양 실적 데이터로 인터랙티브 막대+선 혼합 차트를 Artifact로 만들어줘. 한국어 라벨, 깔끔한 색, 합계 표시. [데이터] (붙여넣기)
    참고 — 이 강의 자료(슬라이드)도 이런 방식으로 만들 수 있습니다.

    5-2Gemini

    구글 생태계와 강하게 연결된 멀티모달 AI

    ⭐강점

    • 구글 문서·시트·Gmail 연동
    • 이미지·표·텍스트 혼합 처리
    • 대용량 자료 분석

    🎯이럴 때

    • 구글 워크스페이스로 일하는 팀
    • 시트 데이터 분석·차트
    • 이미지+문서 같이 다룰 때
    프롬프트이 구글시트의 지역별 청약 경쟁률을 분석해서 상위/하위 지역과 시사점을 정리하고, 비교 차트를 만들어줘.

    5-3Napkin AI

    텍스트를 → 다이어그램·인포그래픽으로 자동 변환

    ⭐강점

    • 글만 넣으면 도식·플로우 자동 생성
    • 여러 시각화 후보 제시
    • 편집·색상·스타일 조정

    🎯이럴 때

    • 제안서·기획서의 개념도
    • 프로세스·전략 한 장 요약
    • 발표용 비주얼 빠르게
    사용법분양 마케팅 추진 프로세스를 문장으로 쓰기 → Napkin이 단계 다이어그램으로 변환 → 색/레이아웃 선택 → 내보내기
    조합 — ChatGPT로 ‘내용’을 만들고 → Napkin으로 ‘그림’을 만든다.

    오늘 핵심 5가지

    • 검색은 ‘찾기’, 생성 AI는 ‘만들기’ — 그래서 ‘시키는 법’이 결과를 가른다
